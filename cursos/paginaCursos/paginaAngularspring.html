CRUD: Angular + Spring

Começar

Sobre o Curso

O curso CRUD Angular + Spring é ideal para quem deseja aprender a desenvolver aplicações completas de ponta a ponta, utilizando o Angular no front-end e o Spring Boot no back-end. Durante o curso, você vai construir um sistema CRUD (Create, Read, Update, Delete), aprendendo a integração entre as duas tecnologias.

O que você vai aprender?

  • Introdução ao projeto
  • Criar a Listagem de Cursos:
    • No Angular
    • No Back-end com Spring.
  • Criar um novo Curso
  • Angular v14 e Refatoração
  • Editar um Curso
  • Excluir um Curso
  • Implementar Validações e Melhorias:
    • Validação de campos.
    • Tratamento de erros.
  • Trabalhar com Relacionamento Um-para-Muitos (Curso-Aulas)
  • Implementar Paginação e adotar Recursos Modernos do Angular
  • Baixar o código-fonte de todos os exemplos vistos em aula.
  • Obter um certificado de conclusão gratuito ao final do curso.
